How offline mode works
1 · Capture offline
Forms, photos & signatures saved to encrypted local store (IndexedDB / SQLite).
2 · Queue
Each record tagged with device ID, GPS & timestamp.
3 · Auto-sync
On reconnect, records POST to the VB.NET REST API with JWT.
4 · Reconcile
Server validates & resolves conflicts (last-write + audit log).
